## 快速安装

运行``./install.sh`` 安装全部字体

windows  运行windowsinstall脚本

------------------

```
    # clone
    git clone https://github.com/zhenruyan/codefont --depth=1
    # install
    cd fonts
    ./install.sh
	windows  运行windowsinstall脚本
    # clean-up a bit
    cd ..
    rm -rf fonts

```
