Quick Installation Script with Mono Font
===============

------------

run ``./install.sh`` install all Mono Font

windows   run  windowsinstall shell

Quick Installation
------------------

```
    # clone
    git clone https://github.com/zhenruyan/codefont --depth=1
    # install
    cd fonts
    ./install.sh
	windows  run windowsinstall shell
    # clean-up a bit
    cd ..
    rm -rf fonts

```
